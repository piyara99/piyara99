### 📌 Featured Projects

🧬 **NSCLC Recurrence Prediction** *(Final-Year Research, Completed)*
- Multimodal architecture combining ResNet-50 image embeddings with structured clinical data using a Gated Multimodal Unit (GMU), evaluated with 5-fold stratified cross-validation
- GMU model achieved 0.6381 AUC (±0.0459); explainability via Grad-CAM, SHAP, and gate weight analysis
- Trained and evaluated on a curated cohort of TCGA-LUAD/LUSC patients, with a Streamlit dashboard for clinical interpretability
- Stack: Python · PyTorch · Scikit-learn
- 🔗 [View Project](https://github.com/piyara99/nsclc-multimodal-project.git)

🧠 **Caregiver Support App** *(Published Research)*
- Flutter + Firebase mobile app with **real-time tracking** for autism caregivers
- Research presented at KDU 18th International Research Conference 2025
- Stack: Flutter · Firebase · Dart
- 🔗 [View Project](https://github.com/piyara99/Autism-Caregiver-App.git)

🏨 **Hotel Room Booking System**
- Concurrent booking platform with **WebSocket-based synchronisation** and conflict prevention
- Built REST APIs handling real-time booking synchronisation across multiple users
- Stack: React · Node.js · MongoDB
- 🔗 [View Project](https://github.com/Ranidu-Ravishka/nova-reservations.git)

🎬 **Cinema Ticket Reservation System**

- Stack: Java · SQL · GlassFish
- 🔗 [View Project](https://github.com/piyara99/CinemaABC.git)
